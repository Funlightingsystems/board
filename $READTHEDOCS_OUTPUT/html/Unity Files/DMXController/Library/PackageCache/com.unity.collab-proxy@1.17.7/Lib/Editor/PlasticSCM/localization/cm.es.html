== CmInvalidCertificateHostname == ATENCIÓN: el nombre de host que aparece en el certificado del servidor no coincide con el nombre de host del servidor. Esto significa que el certificado no ha sido generado para ese nombre de host o que existe algún problema de configuración en el host al que se está conectando.

  • Nombre en el certificado: {0}

  • Nombre del servidor: {1}

Para continuar conectando con este host, elegir ‘Yes’. El proceso de validación

del certificado continuará (no se recomienda).

Para abandonar la conexión, elegir ‘No’ (recomendado).
