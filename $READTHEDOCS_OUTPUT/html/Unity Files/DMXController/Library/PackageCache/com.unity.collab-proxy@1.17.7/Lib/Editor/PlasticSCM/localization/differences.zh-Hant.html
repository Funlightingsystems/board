== Diff == 差異:

== BasePosition == 基底檔案上的位置:

== CompPosition == 比較檔案上的位置:

== OldData == 舊資料:

== NewData == 新資料:

== Type == 類型:

== NullBaseFile == BaseFile 不可為空。

== NullCompareFile == CompareFile 不可為空。

== DiffNotSolutionFound == 找不到解決方案。沒有可計算的差異集合。

== NotValidFileConflictType == 無效的衝突類型。

== NullFilePath == 檔案路徑不可為空。

== NullFileHash == 檔案雜湊不可為空。

== InvalidDifferenceType == 參數類型必須是有效的 differenceType 定義元素。

== TooManyDifferences == 檔案有太多差異，沒有足夠的資源可以完成作業。請嘗試設定不同的差異比較方法來減少差異數量。
